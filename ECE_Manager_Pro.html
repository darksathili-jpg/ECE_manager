Documentation & Conformité

1
Préparer le fichier CSV
Exportez la liste depuis Pronote ou Excel. Le fichier doit contenir exactement 3 colonnes avec la première ligne servant d'en-tête (ignorée automatiquement) :
Nom Prénom ; Classe ; Enseignant
Le séparateur peut être le point-virgule ; ou la virgule ,. L'encodage UTF-8 avec ou sans BOM est supporté.
2
Configurer le jury
Les examinateurs sont extraits automatiquement du CSV. Désactivez les absents en cliquant sur leur badge. Ajoutez des examinateurs externes (n'ayant pas suivi les élèves) via le champ dédié — ils apparaissent avec un bord en pointillé.
3
Règle d'anonymat (contrainte légale)
Conformément à la circulaire n°2019-033 du 22 mars 2019 sur les ECE, un examinateur ne peut pas évaluer un élève qu'il a eu en cours durant l'année. L'algorithme garantit cette règle sur l'ensemble de la répartition. Si aucune solution n'est trouvée après 2000 tentatives, un message d'erreur vous invite à ajouter des examinateurs.
4
Attribution aléatoire des sujets
Chaque examinateur peut disposer de 1 à 4 sujets (configurables dans le panneau gauche via les boutons + / −). Ils sont distribués aléatoirement et de façon équilibrée : si un examinateur interroge 7 élèves avec 3 sujets, la répartition sera au plus ±1 élève par sujet.
5
Annuler et recommencer
Le bouton ✕ à côté du compteur d'élèves réinitialise la sélection sans effacer le jury ni les paramètres. Vous pouvez générer un nouveau planning autant de fois que nécessaire sans recharger le CSV.
Conformité RGPD — Données personnelles
Données traitées : noms, prénoms, classes et enseignants référents d'élèves mineurs (données à caractère personnel selon le RGPD art. 4).

Traitement local uniquement : cette application fonctionne entièrement dans votre navigateur. Aucune donnée n'est transmise à un serveur externe. Aucun cookie de tracking n'est utilisé.

Stockage : seule la liste des examinateurs (sans données élèves) est conservée en localStorage pour faciliter les sessions successives. Les données élèves disparaissent à la fermeture de l'onglet.

Base légale : traitement fondé sur la mission d'intérêt public (art. 6.1.e RGPD) dans le cadre des épreuves nationales. Durée de conservation : session unique.

Recommandation : ne pas laisser l'application ouverte sur un poste partagé. Utiliser un compte utilisateur verrouillé après génération.
